## 🎨 Design Decisions

This isn't a template. Every choice was intentional:

- **Palette** — Warm parchment `#fafaf7` base, deep ink `#0f0f0d`, and a deliberate gold `#c9a84c` pulled from the *"giving wings / उड़ान"* metaphor
- **Typography** — `DM Serif Display` for headlines (character, restrained), `Inter` for body (clarity), `DM Mono` for labels and data (precision)
- **Signature detail** — The nav brand renders as `नये` (Devanagari) + `पंख` (Roman) — bilingual identity at the smallest touchpoint
- **Layout** — Two-column grid throughout: sidebar for context, content area for depth. Mirrors how a well-designed report or magazine feels
- **No JavaScript** — Zero dependencies, zero hydration, zero bundle. Just fast.

## 🛠 Recommended Build — Smart Internship Portal

The single highest-leverage project for NayePankh right now:

```
Intern applies  →  AI assigns task track  →  Dashboard tracks progress
       ↓
Intern submits proof  →  AI validates  →  PDF certificate auto-emails
       ↓
Analytics report generated  →  Shared with donors, NITI Aayog, press
```

**Stack:** `Next.js` · `Firebase` · `Python (FastAPI)` · `Claude API` · `Resend` · `Vercel`
